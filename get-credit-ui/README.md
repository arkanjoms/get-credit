# get-credit-ui

Módulo frontend da aplicação de análise de crédito.

### Tecnologias

Para desenvolvimento deste módulo foram utilizadas as seguintes tecnologias:

 - Vuejs
 - yarn

### Configurando

Após realizar o download do projeto executar os comandos abaixo:

```bash
$ yarn && yarn serve
```

> O webpack está configurado para fazer proxy do serviço backend na porta `8080`

### Acessando aplicação

Após iniciar acessar o navegador na porta `http://localhost:8082`
